## Assignment
Develop a website based on a topic of your choice. The website must meet the following requirements:

### 1. Structure
- The website must consist of at least one fully functional page.
- The page(s) must resemble a cohesive and user-friendly site.

### 2. Content and Features
- Integrate data from at least three different APIs, of which two must be covered in class.

## Submission Details

### 1. Format
- Submit a PDF file containing the written assignment (report).
- Attach a zip file containing all files required for the website (HTML, CSS, JavaScript, Python scripts, etc.).
- Name the main html file as index.html.

### 2. Report Requirements
- For 1 student: Maximum of 10 standard pages.
- For 2 students: Maximum of 16 standard pages.
- Page count excludes the cover page, table of contents, bibliography, and appendices.

This is an internal exam assessed using the 7-point scale in accordance with the curriculum.
